Installation and Setup Instructions
Clone down this repository. You will need node and npm installed globally on your machine.

Nodejs
Installation:
npm install
Create a table in the database:
npx sequelize-cli db:migrate 
To Start Server:
npm run dev

Reactjs
Installation:
npm install
To Start Server:
npm run start
To Visit App
localhost:3000/home
